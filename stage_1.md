# 📋 ЭТАП 1: Окружение, Git и Python-данные (CLI)

🎯 **Цель:** Настроить рабочее место, освоить Git-воркфлоу, закрепить ООП и работу с данными в консольном приложении.

## 🛠️ Настройка

- Установлен Python 3.11+ и добавлен в PATH
- Установлен Git for Windows, проверена команда `git --version`
- Настроен Cursor, включены линтеры `black`/`ruff`
- Создан аккаунт GitHub, настроен SSH/HTTPS доступ
- Создан репозиторий `study-tracker-cli`, клонирован локально
- Настроено виртуальное окружение: `python -m venv .venv`
- Создан `.gitignore` (шаблон Python), добавлены `__pycache__`, `.venv`, `*.pyc`

## 📚 Теория и Практика

- Понятен цикл: `init` → `add` → `commit` → `push`
- Изучены ветки: создание `feature/...`, переключение, слияние через PR
- Освоены `dataclass` или обычные классы для сущностей
- Практика чтения/записи JSON/CSV, обработка `try/except`
- Подключены `numpy`/`pandas`, написаны функции расчёта статистики

## 📦 Проект: Консольный трекер учёбы

- Класс `StudySession` (предмет, время, оценка, заметки)
- Менеджер-класс: добавление, просмотр, удаление, сохранение
- Меню в консоли с валидацией ввода
- Расчёт базовой статистики и запись в отчёт
- Данные сохраняются между запусками (`data.json` или `csv`)
